# Mind Care

**Mind Care** is a student-focused **mental wellness platform** developed as a **hackathon submission**. It helps students manage stress, track emotions, relax, and access support resources through an interactive and user-friendly web application.

Built with **React.js**, the platform focuses on smooth UI/UX, responsiveness, and accessibility.

---

# Features

## 😊 Mood Tracker
- Track your mood throughout the day.
- View mood trends using graphs.
- Understand emotional patterns over time.

## 🤖 AI Chatbot
- Provides calming conversations.
- Suggests breathing exercises.
- Tells jokes to improve mood.
- Supports mental wellness guidance.

## 🌿 Relaxation Station
- Recommended meditation videos.
- Relaxation content for stress relief.
- Helps improve focus and calmness.

## 📚 Articles Section
- Self-help and wellness articles.
- Mental health awareness resources.
- Motivational reading material.

## 📞 Helpline Section
- Emergency and mental health helpline numbers.
- Quick access to support resources.

## Installation

Instructions to set up the project locally:

```bash
# Install Node.js
node -v
npm -v

# Clone the repository
git clone https://github.com/kavyakaira10/Mind-Care.git
cd Mind-Care

# Install Dependencies
npm install

# Install Required Libraries
npm install react-router-dom
npm install chart.js react-chartjs-2
npm install axios
npm install tailwindcss

```
## Usage
After installation, run the following commands:

```bash
# Start Development Server
npm start
```
